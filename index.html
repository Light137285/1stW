1. Differences between LAN; MAN; WAN

  • LAN is a network in a small area such as a home, classroom, or office. It usually has high speed and low cost.
  • MAN is a network that covers a city.
  • multiple LANs together.
  • WAN is a wide network that covers countries or the whole world.
  • The Internet is the biggest example of a WAN.

2. IP Address

  • An IP address is a set of numbers used to identify a device on a network.
  • Every device on the internet needs an IP address.
  • IPv4 is the older version of IP.
  • It has four number groups (e.g., 192.168.0.1) and a limited number of addresses.
  • IPv6 is the newer version.
  • It is longer and provides many more IP addresses.

Packets:

  • A packet is a small piece of data sent over a network.
  • Large data is broken into many packets before being sent.
  • Each packet contains a source address and a destination address.
  • Packets may travel different routes and are reassembled at the destination.
  • Efficient data transfer.
  • Only lost packets need to be resent.
  • Helps networks run faster.
